PRINTER VULNERABILITY SCANNER
Technical Specifications


OBJECTIVE
Need a scanner of vulnerable printers open to access from the Internet.
The result of the scanner is a list of addresses where there is a printer with a confirmed vulnerability.


ALGORITHM
The scanner combines two public services: https://shodan.io and Printer Exploitation Toolkit https://github.com/RUB-NDS/PRET.
- On shodan.io we search, for example, for the word PJL.
- Parsing the results and feeding them to the PRET script.
- If the PRET script was able to join the printer, execute the following commands in sequence
 id
 pwd
 ls
 df
and write the command itself and its result to the log.
- move on to the next search result.

Note: you need an account to access shodan.io. But in idea it is safe even if implemented as a module to the bot.
You can pre-download the necessary data in .csv format and load it separately into bots via configs.
Exporting search results in .csv/.json from shodan.io is a paid service.

REALIZATION
There are two implementation options:
* As a standalone scanner.
  + Execution language - any scripting language, but not too exotic (python, PowerShell, javascript).
  + Quick and easy.
* as a module to the bot
  + implementation is much longer, because you need to rewrite PRET in C++.
  + but you can work in a distributed way
